# Optimize Job Command

Enhance and optimize job descriptions for better candidate attraction.

## Usage
```
/project:optimize-job <job-description-or-file>
```

## Workflow

1. **Analyze Current Description**
   - Extract requirements and responsibilities
   - Identify missing elements
   - Check for bias or exclusionary language

2. **Market Research**
   - Compare with similar roles
   - Analyze salary benchmarks
   - Review competitor postings

3. **Enhancement**
   - Improve clarity and structure
   - Add compelling company benefits
   - Optimize for SEO and job boards
   - Ensure inclusive language

4. **Generate Collateral**
   - Create interview questions
   - Design assessment rubric
   - Build recruitment timeline

## Tools Used
- `mcp__apply-recruitment__analyze_job_requirements`
- `mcp__apply-recruitment__enhance_job_description`
- `mcp__apply-recruitment__get_market_intelligence`
- `mcp__apply-recruitment__generate_interview_questions`

## Options
- `--company-info`: Include company details
- `--optimize-for`: Target specific goals (seo, diversity, clarity)
- `--include-questions`: Generate interview questions
- `--market-data`: Include salary and market analysis

## Example Enhancement

### Before:
"Looking for React developer with 5+ years"

### After:
"Join our innovative team as a Senior React Developer where you'll architect scalable web applications using modern JavaScript technologies. You'll collaborate with designers and backend engineers to deliver exceptional user experiences..."

## Output Includes
- Optimized job description
- SEO keywords
- Diversity metrics
- Interview question bank
- Recruitment timeline
